Security

When you start the ProFume Fumiguide for the first time, you will be asked to provide a "CrypKey". This key allows you entry to use the application. To obtain your CrypKey, you must send an email to Fglcryp@dow.com. Please copy the Site Code that is generated on the opening loading screen into an email message fglcryp@dow.com . The Site Code will be a combination of eighteen numbers and letters. A custom CrypKey will be generated for that unique Site Code. The fumigator will then copy to CrypKey into the appropriate box in the opening loading screen and the installation will then be complete.

If you uninstall the application, you will need a new CrypKey to re-load the application. You will need to email to Fglcryp@dow.com for a new CrypKey.
