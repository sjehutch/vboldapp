Minimum System Requirements

One of the following Microsoft operating systems:

Windows 98
Windows 2000
Windows XP
Windows Vista

The application may run on other Windows-based operating systems (i.e. Windows 95, Windows NT) but is currently not supported.

40 MB minimum available hard disk space.
64 MB RAM minimum
300 MHz Pentium(R) processor (performance level or better) recommended.
1024 x 768 display resolution (at least 256 color recommended). This could be described as High Color (16 bit) setting within the Display Properties Settings screen in Control Panel


Suggested System Specifications

Pentium III Intel processor 500 MHz
10 GB hard disk with 8 GB free space
128 MB RAM (The sample Fumiguide file is 246 KB)
