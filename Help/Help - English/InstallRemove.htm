Installing and Removing the ProFume Fumiguide

To install the application, insert the ProFume Fumiguide disk into the CD-ROM drive, then select and run the Setup.EXE file. You may be asked for information about you and your company. Please complete this information. The Setup program will install the ProFume* Fumiguide* application to the C: drive on your computer. Once the Setup program has finished, you can start the application. Note: You will need a "Site Key" to access the application for the first time. See the Security Section for more information.

If you are reinstalling or updating the application, you can follow the same procedures. Note that you will need to provide a new Site Key to access the application. See the Security Section above for more information.

Important Note! If you re-install the application, you will access ProFume Fumiguide files (*.fum) previously saved. It is a good idea to back-up those Fumiguide files to an external source in the event of serious computer problems (loss or stolen laptop, hard drive crash, etc.).
