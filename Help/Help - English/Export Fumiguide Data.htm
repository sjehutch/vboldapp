Exporting Fumiguide Data and Producing Fumigation Reports

The Fumiguide allows the user to export information into MS-Word using a Report template. With the ProFume Fumiguide desired file open,click on File from the top menu. Then click on "Export" you will see information from fumiguide merging with the "Fumiguide Report.dot" template. This merge may take a few seconds. You will see information automatically being filled in very quickly-that is normal. Once Word is finished creating the new document, you will see the dialog box appear - "The Fumiguide Report was created successfully." Click OK to access the Word document.

The next steps are CRITICAL to not corrupt the report template. You can save the file by clicking on File at the top menu and selecting Save As. DO NOT select Save - this will corrupt your Fumiguide report template.



When the Save As Window opens, use the "Save in:" drop down to select the location on your hard drive where you store your reports. Do NOT use the default “Fumiguide” location that is specified or you will corrupt your document template. Give the file a descriptive and meaningful name. In the "Save as type:" drop down, select Word Document (.doc) from the list, then click the Save button. If a dialog box appears asking if you want to save changes to the “Fumiguide Report.dot” file, click the No button or you will corrupt your document template.



If your *.txt file did not open into the MS-Word document your Macro Security setting may be set too high. To determine your Macro Security setting, within MS-Word go to Tool, Macro and click on Security.



You should select Medium which allows the option to enable or disable macros as previously described.
