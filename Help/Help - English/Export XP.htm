Creating the Fumiguide Report with Windows XP computers

1) Make sure all monitoring readings and all notes on the Customer Info screen have been entered.

2) Click the Save icon at the top of the screen to save your work.

3) From the File menu, select Export.

4) The computer will gather all the information, and place it into a Microsoft Word document.

5) Once Word is finished creating the new document, you will see the following dialog box appear. Click OK to access the Word document.

6) The next steps are CRITICAL to not corrupt the report template.

7) Within MS Word, select Save As… from the File menu. If you select Save you will corrupt your Fumiguide report template.

8) In the resulting dialog box: a. Use the Save in: drop down to select the location on your hard drive where you store your reports. Do NOT use the default “Fumiguide” location that is specified or you will corrupt your document template. b. Give the file a descriptive and meaningful name c. In the Save as type: drop down, select Word Document (.doc) from the list d. Click the Save button.

9) If the following dialog box appears asking if you want to save changes to the “Fumiguide Report.dot” file, click the No button or you will corrupt your document template.

10) You can now modify your Fumiguide report any way you like.
