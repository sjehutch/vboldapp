Requisiti minimi di sistema

Uno dei seguenti sistemi operativi Microsoft:

Windows 98
Windows 2000
Windows XP

L’applicazione può funzionare su altri sistemi operativi su base Windows (es. Windows 95, Windows NT) ma attualmente non è supportata.

40 MB di spazio minimo disponibile sull’hard disk
Minimo 64 MB di RAM
Processore raccomandato Pentium(R) 300 MHz (livello di prestazione o superiore) Risoluzione 1024 x 768 (si raccomanda almeno a 256 colori). Potrebbe essere descritta come impostazione High Color (16 bit) nella schermata delle Impostazioni Proprietà Schermo nel Pannello di Controllo.


Specifiche di sistema suggerite

Pentium III Intel processor 500 MHz
Hard disk da 10 GB con 8 GB di spazio libero
128 MB di RAM (Il file campione di Fumiguide è pari a 246 KB)
