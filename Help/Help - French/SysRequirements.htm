Exigences système minimales

L'un des systèmes d'exploitation Microsoft suivants :

Windows 98
Windows 2000
Windows XP
L'application peut tourner sur d'autres systèmes d'exploitation à base Windows (c'est-à-dire Windows 95, Windows NT), mais ne sont pas supportés actuellement.

40 Mo minimum d'espace disque dur disponible.
64 Mo RAM minimum.
Processeur Pentium® 300 MHz (niveau de performances ou mieux) recommandé.
Affichage résolution 1024 x 768 (au moins 256 couleurs recommandé). Ceci est le paramétrage Couleurs moyennes (16 bits) dans l'écran Paramètres des Propriétés d'affichage du Panneau de configuration.


Spécifications système suggérées

Processeur Intel Pentium III 500 MHz
10 Go espace disque avec 8 Go d'espace libre
128 Mo RAM (Le fichier échantillon Fumiguide occupe 246 Ko)
