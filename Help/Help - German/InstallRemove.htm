Installieren und Entfernen der ProFume Fumiguide Software

Um das Programm zu installieren, legen Sie die ProFume Fumiguide CD in das CD-ROM-Laufwerk ein und rufen Sie dann die Setup.EXE Datei auf. Hier werden Sie eventuell nach Angaben zu Ihrer Person und zu Ihrem Unternehmen gefragt. Bitte geben Sie die entsprechenden Informationen ein. Das Setup-Programm installiert die ProFume* Fumiguide* Software auf Laufwerk C: Ihres Computers. Nach Abschluss des Setup-Programms können Sie die Anwendung starten. Hinweis: Beim erstmaligen Aufrufen des Programms benötigen Sie für den Zugang einen so genannten "Site Key". Nähere Informationen hierzu finden Sie im Abschnitt „Sicherheit / CrypKey“.

Bei der Neuinstallation oder Aktualisierung der Software können Sie genauso vorgehen. Bitte beachten Sie hierbei, dass Sie für den Zugang zum Programm einen neuen Site Key benötigen. Nähere Informationen hierzu finden Sie oben im Abschnitt „Sicherheit / CrypKey“.

Wichtiger Hinweis! Bei der Neuinstallation der Software werden Sie auf zuvor gespeicherte ProFume Fumiguide Dateien (*.fum) zugreifen. Es empfiehlt sich, diese Fumiguide-Dateien auf externe Datenträger zu speichern, für den Fall, dass gravierende Probleme auftreten (Verlust oder Diebstahl des Laptops, Absturz der Festplatte usw.).
