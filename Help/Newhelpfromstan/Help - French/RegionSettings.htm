Changer la configuration régionale pour afficher l'heure correctement

En fonction des paramètres dans le Panneau de configuration du système d'exploitation Microsoft Windows, les fichiers affichant l'horaire peuvent indiquer un horaire incorrect. Dans la capture d'écran ci-après, notez que les champs d'horaire n'affichent ni am ni pm.



Pour corriger ceci, changez les paramètres d'affichage de l'horaire. Cliquez sur le bouton Démarrer et sélectionnez Paramètres, puis Panneau de configuration.



Dans le Panneau de configuration, double cliquez sur l'icône Paramètres régionaux.



Ensuite, cliquez sur l'onglet Heure dans les Propriétés d'affichage.



Sélectionnez AM pour le symbole AM et PM pour le symbole PM.



Cliquez sur OK pour enregistrer les paramètres.

Lors de la prochaine exécution de Fumiguide, vous devriez voir apparaître AM/PM.
