Sécurité

Lors du premier démarrage de ProFume Fumiguide, vous êtes invité à saisir un code d’accès. Ce code débloque l'utilisation de l'application. Pour obtenir votre code d'accès, vous devez envoyer un Email à Fglcryp@dow.com accompagné du code machine qui est généré dans l'écran d'accueil lors du démarrage du programme. Le code machine sera une combinaison de dix-huit chiffres et lettres. Un code d’accès personnalisé sera généré pour ce code machine unique. Le fumigateur copie ensuite le code d’accès dans la case correspondante de l'écran d’accueil, et ainsi l'installation sera complète.

Si vous désinstallez le programme, vous aurez besoin d'un nouveau code d'accès pour redémarrer l'application. Vous devrez à nouveau envoyer un Email à Fglcryp@dow.com
