Installer et désinstaller ProFume Fumiguide

Pour installer l'application, insérez le disque Le Fumiguide ProFume dans le lecteur de CD-ROM, ensuite sélectionnez et démarrez le fichier Setup.EXE. Le programme peut vous inviter à saisir des informations sur vous et votre entreprise. Saisissez ces informations. Le programme Setup installe l'application ProFume* Fumiguide* sur le lecteur C: de votre ordinateur. Une fois le programme Setup terminé, lancez l'application. Remarque : Vous aurez besoin d’un code d’accès pour accéder à l'application lors de sa première exécution. Pour davantage d'informations, consultez la section Sécurité/Code d’accès.

Si vous réinstallez ou mettez à jour l'application, utilisez les mêmes procédures. Veuillez noter que vous ne devrez pas fournir de nouvelle Clé de Site pour accéder à l'application. Pour davantage d'informations, voir la section Sécurité.

Remarque importante ! Si vous réinstallez l'application, vous accéderez aux fichiers Fumiguide ProFume (*.fum) enregistrés auparavant. Il serait judicieux de sauvegarder ces fichiers Fumiguide sur une source externe pour palier à tout problème informatique grave (perte ou vol de PC portable, panne de disque dur, etc.).
