Systemanforderungen

Eines der folgenden Microsoft-Betriebssysteme:

Windows 98
Windows 2000
Windows XP

Diese Anwendung läuft unter Umständen auch unter anderen Windows-basierten Betriebssystemen (z. B. Windows 95, Windows NT), diese werden jedoch derzeit nicht unterstützt.

Mindestens 40 MB freier Festplattenspeicher
Mindestens 64 MB Arbeitsspeicher (RAM)
300 MHz Pentium(R) Prozessor (oder höher) empfohlen
1024 x 768 Bildschirmauflösung (mindestens 256 Farben empfohlen). Eventuell beschrieben als Einstellung High Color (16 Bit) unter Bildschirmeigenschaften / Einstellungen an der Steuerkonsole.

Empfohlene Systemspezifikation

Pentium III Intel Prozessor 500 MHz
10 GB Festplatte mit 8 GB freiem Speicherplatz
128 MB Arbeitsspeicher (RAM) (Die Fumiguide Musterdatei umfasst 246 KB)
