Sicherheit / CrypKey

Wenn Sie das ProFume Fumiguide Programm zum ersten Mal aufrufen, werden Sie zur Eingabe eines so genannten "CrypKey" aufgefordert. Hierbei handelt es sich um eine Art Schlüssel, der Ihnen den Zugang zur Nutzung dieses Programms ermöglicht. Den CrypKey können Sie unter Fglcryp@dow.com anfordern. Hierzu kopieren Sie bitte den auf dem Ladebildschirm erzeugten Site Code (Zugangs­code) in eine E-Mail-Nachricht an Fglcryp@dow.com . Der Site Code besteht aus einer Kombination aus achtzehn Ziffern und Buchstaben. Für jeden Site Code wird jeweils ein individueller CrypKey erzeugt. Anschließend kopiert der Begasungsleiter den CrypKey in das entsprechende Feld im Ladebildschirm. Die Installation ist damit abgeschlossen.

Wenn Sie das Programm deinstallieren, wird anschließend zum erneuten Laden ein neuer CrypKey benötigt. In diesem Fall wenden Sie sich bitte an Fglcryp@dow.com, um einen neuen CrypKey zu erhalten.
