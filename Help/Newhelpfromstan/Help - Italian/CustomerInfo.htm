Tab Informazioni sul Cliente

Aprendo un nuovo file Fumiguide* per il gas fumigante ProFume* si visualizzer‡ una schermata come quella seguente. Sebbene non sia obbligatorio inserire nessun dato, nel tab Informazioni sul Cliente si possono comodamente memorizzare recapiti, dettagli del contatto e annotazioni varie relative alla fumigazione. La sezione Note, situata nella parte inferiore della schermata, Ë riservata alle annotazioni di una certa lunghezza, quali la descrizione di impostazioni specifiche, di procedure di sigillatura o di notificazione pubblica. Queste annotazioni saranno salvate nel file per riferimento futuro.
