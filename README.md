# VBOldApp
VB Old APP - A Classic Visual Basic .NET Application

## Description
This is a simple VB.NET console application that demonstrates classic Visual Basic programming features including:
- Console output and user interaction
- Basic arithmetic calculations
- String manipulation
- Date/time handling
- Modular programming with subroutines

## Prerequisites
- .NET 9.0 SDK or later
- Compatible with Windows, macOS, and Linux

## Building the Application
To build the application, run the following command in the repository root:

```bash
dotnet build
```

## Running the Application
To run the application, use one of the following commands:

### Option 1: Run from solution root
```bash
dotnet run --project VBOldApp
```

### Option 2: Run from project directory
```bash
cd VBOldApp
dotnet run
```

## Features Demonstrated
1. **User Interface**: Console-based application with formatted output
2. **Calculator Demo**: Basic arithmetic operations (addition, subtraction, multiplication, division)
3. **String Demo**: String manipulation including length, case conversion, and concatenation
4. **Date/Time**: Display current date and time
5. **User Interaction**: Wait for user input before exiting

This application showcases traditional VB.NET programming patterns and serves as a simple example of a "classic" Visual Basic application structure.
